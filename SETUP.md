# Инструкция по запуску проекта

## Шаги для запуска

### 1. Установка зависимостей
```bash
npm install
```

### 2. Настройка базы данных
```bash
# Генерация Prisma клиента
npx prisma generate

# Создание базы данных
npx prisma db push
```

### 3. Создание переменных окружения
Создайте файл `.env.local` в корне проекта:
```bash
cp env.example .env.local
```

Отредактируйте `.env.local` и добавьте ваши настройки.

### 4. Запуск проекта
```bash
npm run dev
```

Проект будет доступен по адресу: http://localhost:3000

## Первоначальная настройка

### Создание администратора
После запуска проекта, вам нужно создать первого администратора. Для этого:

1. Зарегистрируйтесь через форму регистрации
2. В базе данных измените роль пользователя на ADMIN:
```sql
UPDATE User SET role = 'ADMIN', status = 'ACTIVE' WHERE email = 'your-email@example.com';
```

### Создание токенов для регистрации
Для регистрации новых пользователей нужно создать токены:
```sql
INSERT INTO User (email, password, token, role, status) 
VALUES ('temp@example.com', 'temp', 'TOKEN123', 'USER', 'PENDING');
```

Затем используйте токен 'TOKEN123' для регистрации.

## Структура базы данных

После выполнения `npx prisma db push` будет создана база данных SQLite с таблицами:

- **User** - пользователи системы
- **Wallet** - кошельки пользователей  
- **NetworkPair** - связки сетей
- **Payment** - транзакции
- **Referral** - реферальная система
- **SupportTicket** - обращения в поддержку
- **Session** - сессии пользователей

## Проблемы и решения

### Ошибка "Module not found"
```bash
npm install
```

### Ошибка базы данных
```bash
npx prisma generate
npx prisma db push
```

### Проблемы с типами TypeScript
```bash
npm install --save-dev @types/node @types/react @types/react-dom
```
